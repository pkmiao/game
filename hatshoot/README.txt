====================================================
  帽孩出击 Hat Shoot —— 玩法说明
====================================================

【怎么开始玩】
  方法一（最简单）：双击 index.html，点"开始游戏"。
  方法二（备用）：双击 start.bat（电脑装有 Python 时会自动
  打开浏览器；没有 Python 就用方法一）。

  游戏完全离线，不需要联网。

【目标】
  楼顶上一共有 12 个黑手党打手，全部消灭就获胜！
  被打倒或掉下楼顶扣血，血没了就失败。脱战 5 秒会自动回血。

【键位】
  W A S D ...... 移动
  鼠标 ......... 视角
  左键 ......... 开火
  右键（按住）.. 瞄准
  Shift ....... 冲刺
  空格 ......... 跳跃
  C ........... 蹲下；冲刺中按 C = 滑铲
  R ........... 换弹
  1 / 2 ....... 激光枪 / 榴弹发射器
  F ........... 雨伞近战
  G ........... 投掷药瓶手雷（毒雾持续伤害）
  V ........... 切换第一 / 第三人称
  Esc ......... 暂停

【想改难度？】
  用记事本打开 js\config.js，里面的数字都能改：
  敌人数量（count）、敌人伤害（damage）、你的血量（hp）、
  武器伤害等等，改完保存再刷新游戏即可。
